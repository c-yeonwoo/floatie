# App Store / Play Store 카피 초안 — 쪽지 (임시명)

> 포지션: 라이트 소셜 / 설렘. “데이트·연애·매칭·이상형” 금지.  
> 제품명 **쪽지**는 작업용. 최종 브랜드 확정 시 전면 치환.

## 앱 이름

쪽지

## 부제 (subtitle)

익명 미션이 도착하면, 답이 괜찮았을 때만 서로를 연다

## 짧은 설명

모르는 사람이 보낸 가벼운 미션에 답하세요. 서로 OK면, 그때 프로필이 열립니다.

## 키워드 (초안)

익명쪽지,미션,설렘,소셜,쪽지,익명질문,OK

## 긴 설명

쪽지는 익명의 미션과 질문이 도착하는 가벼운 소셜 앱입니다.

누군가 보낸 카드에 짧게 답해 보세요. 오늘 기분 한 단어, 점심 인증, 취향 질문이면 충분합니다. 사진 미션은 아직 없어요.

보낸 사람과 받은 사람이 둘 다 “괜찮았다”고 할 때만 미니 프로필이 열립니다. 한쪽만 OK면 정체는 계속 비밀입니다. 패스해도 페널티는 거의 없습니다.

열린 뒤에는 짧은 대화를 나눌 수 있어요. 외부 연락은 서로 원할 때만.

차단·신고·하루 수신 한도로 안전하게 즐기세요. 만 18세 이상.

## What’s New (첫 출시)

- 익명 미션 송수신 (텍스트·칩)
- 양방향 OK 시 프로필 unlock
- 짧은 인앱 대화
- 차단·신고·안전 센터

# 미션 프리셋 톤 가이드 + 시드

> 런칭 목표: **80개** (질문 50 · 행동인증 30). 아래는 톤 규칙과 시드 샘플(~30). 나머지는 같은 규칙으로 확장.

## 톤 원칙

1. **가벼움** — 30초~하루 안에 답/수행 가능  
2. **수치심 최소** — 외모 품평, 연봉, 성적, 신상 추궁 금지  
3. **양방향 재미** — 보낸 사람도 “답 보고 싶다”가 나게  
4. **칩 친화** — 가능하면 3~5개 칩 답변 옵션을 프리셋에 붙임  
5. **시점** — “오늘/요즘”을 자주 써서 일일 루프와 맞춤  

## 금지 (프리셋·커스텀 공통)

- 성적·야한·신체 부위 요구  
- 만남 강요, 장소 집합, 주소/학교명 요구  
- 혐오·차별·자해·불법  
- 타 유저 사칭, 개인정보 교환 유도 (unlock 전)  
- “이상형/키/스펙” 데이팅 설문형

`mood` `food` `hobby` `daily` `taste` `tiny_dare` `memory` `comfort`

## 커스텀 미션 규칙

- 최대 40자  
- 금칙어 리스트 + 경량 분류기(성적/만남/혐오)  
- 실패 시: “이 미션은 보낼 수 없어요. 프리셋을 써보세요”  
- 반복 거부 유저: 커스텀 24h 잠금  

## Phase 2 사진 미션 (참고만, MVP Out)

- 손, 오늘의 신발, 책상 모서리 등 **얼굴 비필수** 프롬프트 우선  
- 옵트인 유저끼리만  
- 기본 blur 토글, 신고 시 즉시 숨김

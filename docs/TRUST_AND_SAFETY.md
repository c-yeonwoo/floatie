# Trust & Safety — BottleNote

## 원칙

1. Unlock 전 신원 최소화  
2. 패스는 쉽고, 차단은 확실  
3. 핵심 루프(수신·OK)는 유료·광고로 오염하지 않음  
4. 만 18+만  

## 가입·연령

- 출생연도 입력 + 만 18 미만 가입 거부  
- 허위 연령 신고 시 계정 검토·정지  

## 미션 모더레이션

| 경로 | 조치 |
|------|------|
| 프리셋 | 운영 화이트리스트만 |
| 커스텀 | 금칙어 → 분류기 → (애매) 수동 큐 | 
| 신고 | 해당 미션/유저 즉시 숨김 후보 + 24h SLA |

## 차단·신고

- 차단: 상호 전달·검색·채팅 불가, 쿨다운과 별개로 영구  
- 신고 사유: 불쾌 미션, 혐오, 스팸, 성적, 미성년 의심, 기타  
- 누적 신고: 송신 제한 → 정지  

## 성비·남용 가드

- 일일 수신 캡 (기본 8)  
- 일일 송신 소프트 캡 (기본 3, MVP 무료)  
- 성비 대기열 밸런싱  
- 동일 쌍 14일 쿨다운  

## 채팅 안전

- unlock 스레드만 채팅 가능  
- 외부 연락처는 양측 동의 후에만 표시  
- 채팅 내 신고/차단 동일 진입점  

## 안전 센터 (인앱)

- 가이드: “Unlock 전엔 개인정보 넣지 마세요”  
- 긴급: 차단, 신고, 계정 비활성화  
- 연락: 지원 이메일 (런칭 전 확정)  

## Phase 2 사진

- 옵트인, face-centric 프롬프트 지양, blur, 선제 스캔(가능 시)  
- 위반 시 사진 기능 영구 박탈 가능

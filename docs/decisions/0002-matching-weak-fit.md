# ADR 0002 — 매칭: 약한 fit

- Status: Accepted
- Date: 2026-07-14

## Context

완전 랜덤은 성비·연령 미스매치로 이탈이 크고, 딥 취향 fit은 데이팅앱과 겹친다.

## Decision

**가드형 랜덤(약한 fit).** 하드 필터(연령·성별 선호·차단·쿨다운·수신 캡) + 소프트 스코어(지역·연령대·활성·성비 큐).

MVP에서 ML 취향·얼굴·스펙 fit은 하지 않는다. 동일 쌍 재매칭 쿨다운 14일.

## Consequences

- 구현이 설명 가능하고 디버깅하기 쉽다.
- 전국 폴백 시 “먼 거리” 불만이 생길 수  → UI에 지역 힌트(unlock 후)로 완화.

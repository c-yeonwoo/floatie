# ADR 0001 — 카테고리: 라이트 소셜 / 설렘

- Status: Accepted
- Date: 2026-07-14

## Context

익명 미션 → 양방향 OK → 프로필 unlock 루프는 설렘을 주지만, 스토어·정책·카피에서 **데이팅 앱**으로 잡을지 **라이트 소셜**로 잡을지에 따라 리뷰 난이도·타깃·KPI가 갈린다.

## Decision

**라이트 소셜 / 설렘**으로 포지션한다. 스토어 카테고리는 Social Networking. 카피에서 데이트·연애·매칭·이상형 언어를 쓰지 않는다.

Unlock 후 대화·연락은 허용하되, 북극성 KPI는 주간 양방향 OK 수다.

## Consequences

- 안전/연령 게이트는 유지하되, 데이팅 전용 규제 프레임을 과도하게 끌어오지 않는다.
- 나중에 데이팅으로 피벗하면 스토어 메타·온보딩 전면 수정이 필요하다 (의도적 트레이드오프).
